# Ksyusha Antistress Updates

Эти файлы нужно загрузить в публичный GitHub Pages репозиторий `ksyusha-antistress-updates`.

Проверочный URL после настройки Pages:

```text
https://shishshlov-sys.github.io/ksyusha-antistress-updates/ksyusha-update.json
```

Файлы:

- `ksyusha-update.json` - манифест обновления.
- `KsyushaAntistress-install.apk` - APK, который скачает приложение.
- `.nojekyll` - отключает Jekyll-обработку GitHub Pages.
